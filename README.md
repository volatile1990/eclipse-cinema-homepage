# Eclipse Cinema Homepage

Statische Website fuer GitHub Pages.

## Lokal testen

1. Repository oeffnen.
2. `index.html` im Browser starten.

## GitHub Pages veroeffentlichen

1. Repository zu GitHub pushen.
2. In GitHub: `Settings` -> `Pages`.
3. Bei `Build and deployment` `Deploy from a branch` auswaehlen.
4. Branch `main` (oder `master`) und Ordner `/ (root)` waehlen.
5. Speichern und auf die Bereitstellung warten.

Die Seite nutzt lokale Assets unter `assets/images` und benoetigt keinen Build-Prozess.
